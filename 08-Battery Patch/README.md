# 电池热补丁收录及方法指导

## 电池热补丁收录

本项目收录了部分笔记本的 OpenCore 电池热补丁(如 ThinkPad 系列笔记本)，供大家参考使用

更多已经制作好的电池热补丁可参考 GZ小白 的收录项目

项目页面：<https://github.com/GZXiaoBai/Hackintosh-Battery-Hotpatch>
